# Physical Store – Backend (v1)

Backend de uma aplicação de Physical Store desenvolvido em JavaScript
utilizando Node.js, com foco na criação de APIs e implementação
de regras de negócio.

## Tecnologias
- JavaScript
- Node.js
- Express
- Git / GitHub

## Objetivo do projeto
Praticar conceitos de desenvolvimento backend, incluindo estruturação
de rotas, organização de código e integração de APIs.
